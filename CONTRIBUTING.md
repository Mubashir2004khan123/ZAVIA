# Contributing to ZAVIA

Thank you for your interest in contributing to ZAVIA! We appreciate your help in making this project better.

## What is ZAVIA?

ZAVIA is a domain-specialized Large Language Model designed to explain Indian laws clearly, accurately, and responsibly. It covers:
- Constitution of India
- IPC (Indian Penal Code)
- CrPC (Code of Criminal Procedure)
- CPC (Code of Civil Procedure)
- Indian Contract Act
- Transfer of Property Act
- Bharatiya Nyaya Sanhita (BNS)
- Core legal concepts & structured explanations

## How to Contribute

### 1. Fork the Repository
Click the "Fork" button on GitHub to create your own copy of the repository.

### 2. Clone Your Fork
```bash
git clone https://github.com/YOUR-USERNAME/ZAVIA.git
cd ZAVIA
```

### 3. Create a Feature Branch
```bash
git checkout -b feature/your-feature-name
```

### 4. Make Your Changes
- Add or improve documentation
- Enhance legal explanations
- Add new legal topics
- Fix bugs or improve code quality

### 5. Commit Your Changes
```bash
git commit -m "Add/Fix: Brief description of your changes"
```

### 6. Push to Your Fork
```bash
git push origin feature/your-feature-name
```

### 7. Submit a Pull Request
- Go to the original repository on GitHub
- Click "New Pull Request"
- Select your branch and provide a clear description of your changes

## Guidelines for Contributions

### Documentation
- Keep explanations clear and accurate
- Use proper formatting and markdown
- Include examples where relevant
- Reference legal sections appropriately

### Code Quality
- Follow consistent coding standards
- Write clear comments for complex logic
- Test your changes thoroughly

### Legal Accuracy
- Ensure all legal information is accurate
- Cite relevant laws and sections
- Note if information is specific to a particular jurisdiction
- Include dates if laws have been updated

## Types of Contributions We Welcome

- 📚 Documentation improvements
- 📖 New legal explanations or topics
- 🐛 Bug fixes
- ✨ Feature enhancements
- 📝 Content updates and corrections
- 🔍 Accuracy reviews

## Reporting Issues

If you find an error or have a suggestion:
1. Check if the issue already exists
2. Create a new issue with a clear title and description
3. Include relevant details and examples

## Code of Conduct

- Be respectful and inclusive
- Provide constructive feedback
- Focus on the idea, not the person
- Help others learn and grow

## Questions?

Feel free to open an issue if you have any questions about contributing!
